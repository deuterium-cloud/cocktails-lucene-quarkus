# Cocktails Lucene Demo Project

This is a demo project to try Lucene search library with quarkus framework. 

## Running the application
Firstly, you need to compile the app:
```shell script
./mvnw package
```
Run the app:
```shell script
java -jar target/cocktails-dev.jar
```
